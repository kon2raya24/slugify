# slugify

[![npm version](https://img.shields.io/npm/v/slugify.svg)](https://www.npmjs.com/package/slugify)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Lightweight URL slug generator with Unicode support

## Installation

```bash
npm install slugify
```

## Functions

- `slugify` — Convert string to URL-friendly slug
- `deslugify` — Convert slug back to readable string
